# 🏏 FieldUp

**Elevating Grassroots Cricket with AI Intelligence.**

FieldUp is an AI-powered cricket intelligence platform designed specifically for local teams, academies, and aspiring athletes. We bring professional-grade analytics, tactical insights, and injury prediction to the grassroots level—no million-dollar infrastructure required.

---

## 🚀 Key Features

*   **🧠 AI Match Strategy Generator:** Get optimal lineup suggestions, batting orders, and bowling plans based on player form and match conditions.
*   **🩺 Injury Risk Predictor:** Track workload and fatigue to predict burnout and keep players in peak condition.
*   **📊 AI Performance Analyzer:** Deep-dive into batting consistency, bowling economy, and strike rate trends with AI-generated coaching insights.
*   **📋 Match Intelligence:** Simplified match logging and scorecard management for local tournaments.

---

## 🛠 Tech Stack

### Backend & AI
- **Node.js & Express:** Robust backend orchestration.
- **FastAPI (Python):** High-performance ML service for injury and outcome prediction.
- **Mongoose & MongoDB:** Flexible data modeling for complex cricket statistics.
- **Gemini AI:** Advanced LLM integration for natural language tactical coaching.

### Frontend
- **Next.js & TypeScript:** Modern, type-safe user interface.
- **Tailwind CSS & Shadcn UI:** Sleek, responsive design for coaches and players on the go.

## ⚙️ Quick Start

### Prerequisites
- Node.js (v18+)
- MongoDB
- Python (v3.9+)

### Backend Setup
1. `cd backend`
2. `npm install`
3. Rename `.env.example` to `.env` and add your `MONGO_URI`.
4. `npm run dev`

### AI Service Setup
1. `cd ai-service`
2. `pip install -r requirements.txt`
3. `uvicorn app:app --reload`
